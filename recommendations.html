Considerations and Recommendations

uBlock Origin (unwanted content blocker)

In order to block a lot of unwanted content, such as tracking and malvertising, I usually install uBlock Origin.
This extension is a 'wide-spectrum blocker' which has proven to be very versatile and effective.

  • uBlock Origin - Google Chrome, Microsoft Edge (Chromium), Brave Browser
  • uBlock Origin - Mozilla Firefox
  • (uBlock Origin - Alternate link for Microsoft Edge (Chromium))

 

Bitwarden - Open Source Password Manager

For managing all passwords, secure notes, and to maintain a good attitude of 'password hygiene', I use Bitwarden.
Bitwarden is available as both a paid and free plan. The paid plan offers a few welcome extras.

  • Bitwarden - Home Page
  • Bitwarden - Download

 

Total Commander - File Manager (for Windows)

For easy but comprehensive file management, Total Commander can be considered a 'Swiss knife' of file management.
It also offers 'side by side' comparison of two files, a feature which I use a lot for creating the different scripts of UltimateKEYS.

  • Total Commander (shareware)

 

IrfanView (and XnView) - Image Viewers

As comprehensive image viewers, Irfanview (and XnView) can be considered the 'Swiss knives' of image viewers, supporting a wide range of formats.
Overall, IrfanView feels somewhat a bit snappier, although only XnView MP has support for AVIF at this time...

  • IrfanView (free for personal, non-commercial use, shareware for commercial purposes)
  • XnView (free for personal, non-commercial use, shareware for commercial purposes)

 

Antivirus Software (for Windows)

In the past, I used different antivirus products on Windows, but since 2017, I removed all third-party products. Now I rely entirely on Windows 10's built-in security (Windows Defender, default in Windows 10/8.1).

This is also the counsel from a former Firefox developer, not to install any third-party security software on Windows 10.

Windows 10 (and 8.1) then automatically activates its built-in 'Windows Defender'. This antivirus is very stable and also turns out to be the most reliable integration concerning updates (and upgrades) for Windows itself.
