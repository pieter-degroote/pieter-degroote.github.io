Considerations and Recommendations

uBlock Origin (unwanted content blocker)

In order to block a lot of unwanted content, such as tracking and malvertising, I usually install uBlock Origin.
This extension is a 'wide-spectrum blocker' which has proven to be very versatile and effective.

  • uBlock Origin - Mozilla Firefox
  • uBlock Origin Lite - Google Chrome, Chromium, Microsoft Edge
  • uBlock Origin Lite - Alternate link for Microsoft Edge

=> Note that uBlock Origin works best on Firefox. On Chrome, Chromium and Edge, you can use uBlock Origin Lite instead.

=> Also note that only one content blocker may (or should) be activated in the browser at a time.

 

Bitwarden - Open Source Password Manager

For managing all passwords, secure notes, and to maintain a good attitude of 'password hygiene', I use Bitwarden.
Bitwarden is available in both a paid and a free plan. The paid plan offers a few welcome extras, like enhanced Two-Factor Authentication (2FA).

  • Bitwarden - Home Page
  • Bitwarden - Download

 

Antivirus Software (for Windows)

In the past, I used different antivirus products on Windows. Since 2017, I use its built-in security.

For a second opinion, in case of doubt, I use VirusTotal, which allows uploading a file for scanning against roughly 67 different reputable antivirus/antimalware engines.

  • Website :  VirusTotal.com
